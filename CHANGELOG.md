# Changelog

All notable changes to this project will be documented in this file.

This component is part of [ThorsAnvil](https://github.com/Loki-Astari/ThorsAnvil). See the [parent changelog](https://github.com/Loki-Astari/ThorsAnvil/blob/master/CHANGELOG.md) for full release history.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [10.0.5] - 2026-05-16
### Added
- Slash command acknowledge (Ack) with extended return values
- View support: `viewPush()`, `viewUpdate()`, and modal handling
- User action handlers with value extraction from controls
- Slash command handling
- Event handlers for all Slack event types
- Simple NisseBolt::App framework for building Slack bots
- Validation logic for Slack request verification

### Changed
- Improved BlockKit type names
- Refactored value handling in views
- Simplified Visitor pattern for event dispatch
- Simplified Checkbox handling
- Improved logging

## [9.1.14] - 2026-04-13

### Changed
- Added README
- Removed dependency on Mug
- Made submodules relative
